About Paulo

Software Developer currently working for UNC Lineberger Comprehensive Cancer Center

I currently work as a remote full stack web developer with a contract that ends soon. I'm tasked with developing a JavaScript stack web application that emulates Lineberger's "Laboratory Information Management System" or LIMS. LIMS was written in Java and connects to an Oracle DB, and so my backend tasks include writing server-side script in Node.js to interact with that same Oracle DB. In the front end, I use JQuery extensively with Object-Oriented JavaScript practices. In the testing side, I use Mocha and Chai to write unit tests.

I graduated last June from UC San Diego and I was an Electrical Engineering major for my first two years as a student. I made the switch to Cognitive Science and Computer Science at the end of my sophomore year. Besides web development, I do Machine Learning & Data Science programming in Kaggle. I lived there for 5 years, moving to New Jersey for half a year, and finally settling in Ventura County here in California. Some of my hobbies include reading, cooking, and sports. Actor Graphs

Using Hash tables, Heaps, and Trees, I made a GUI-less version of the game "Six Degrees of Kevin Bacon". It makes a graph data structure that uses Movie actors as vertices and films as edges.

Autocomplete

Using Multiway Tries, Priority Queues, and Stacks, I was able to make my own AutoComplete program that is prominent in every search service. It uses Depth First Search to predict word and phrase completions.
