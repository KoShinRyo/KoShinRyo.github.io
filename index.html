Actor Graphs

Using Hash tables, Heaps, and Trees, I made a GUI-less version of the game "Six Degrees of Kevin Bacon". It makes a graph data structure that uses Movie actors as vertices and films as edges.

Autocomplete

Using Multiway Tries, Priority Queues, and Stacks, I was able to make my own AutoComplete program that is prominent in every search service. It uses Depth First Search to predict word and phrase completions.

Huffman Compression
